# aliqrcode
自动批量生成支付宝收款码工具

# 技术详述
Android里有一个叫`AccessibilityService`的服务，可以模拟点击输入等事件。具体可参考csdn上微信红包外挂实现（http://blog.csdn.net/feifei454498130/article/details/43891781/）
